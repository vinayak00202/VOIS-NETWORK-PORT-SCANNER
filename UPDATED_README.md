## 📌 Project Overview

The **Network Port Scanner** is a lightweight cybersecurity tool designed to scan a target system and identify **open, closed, and filtered ports**. It helps detect potential vulnerabilities and improves system security by analyzing active network ports.

This project uses **Python socket programming** to establish connections with ports and determine their status in real time.

---

## ⚠️ Problem Statement

In today’s digital world, systems are exposed to multiple network vulnerabilities.

- Open ports can be exploited by attackers  
- Manual port checking is time-consuming  
- Lack of automation reduces efficiency  

👉 Therefore, an automated solution is required to **quickly and accurately detect open ports** and enhance system security.
